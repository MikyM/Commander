Interface ICommandHandlerResolver

A resolver for command handlers utilizing injected current -scoped- System.IServiceProvider to resolve optional handlers (like System.Lazy`1).

Use this only when for some reason you don't want to utilize System.Lazy`1 and when dealing with optional dependencies which creation is expensive.
